# `<initialism>`

The `<initialism>` tag belongs in [`<docinfo>`](./docinfo.mdx) and gives
a short identifier for the whole document. PreTeXt uses it in some
generated outputs, such as Sage worksheet titles, and it is different
from the inline [`<init>`](./init.mdx) tag.
