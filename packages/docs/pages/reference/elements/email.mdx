# `<email>`

The `<email>` tag gives an email address for a person listed in metadata,
such as an [`<author>`](./author.mdx) or [`<editor>`](./editor.mdx). In HTML
output it is typically rendered as a clickable `mailto:` link.

Use `<email>` as part of a person's contact information rather than as normal
body text.
