# `<species>`

The `<species>` tag gives the species portion of a scientific name
inside a [`<taxon>`](./taxon.mdx). It is usually paired with
[`<genus>`](./genus.mdx) so the organism name is marked up
semantically.

Use `<species>` for biological classification rather than ordinary
italicized text.

<SyntaxDisplay name="species" />

## Examples

```ptx-example
<FRAGMENT parents="p section subsection">
    The gray wolf is classified as
    <taxon><genus>Canis</genus> <species>lupus</species></taxon>.
</FRAGMENT>
```
