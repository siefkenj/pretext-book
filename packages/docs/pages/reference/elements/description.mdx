# `<description>`

The `<description>` tag provides a structured long description for an
[`<image>`](./image.mdx). It is primarily used for accessibility so readers
who cannot see the image can still access its important content.

Unlike [`<shortdescription>`](./shortdescription.mdx), a `<description>` can
contain one or more paragraphs and other structured content such as a table.

## Examples

```ptx-example
<FRAGMENT parents="section subsection figure">
    <image source="graphs/parabola.png" width="50%">
        <description>
            <p>
                A graph of a downward-opening parabola with vertex at
                <m>(1,3)</m>.
            </p>
        </description>
    </image>
</FRAGMENT>
```
