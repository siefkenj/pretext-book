# `<program>`

The `<program>` tag displays a source-code listing, usually with syntax
highlighting based on its `language` attribute. The code itself may appear
directly or inside a [`<code>`](./code.mdx) child.

A program can also include pieces such as [`<preamble>`](./preamble.mdx),
[`<postamble>`](./postamble.mdx), [`<tests>`](./tests.mdx), or
[`<stdin>`](./stdin.mdx) to describe setup and execution details.
