# `<demonstration>`

The `<demonstration>` tag is a block for an interactive
demonstration or guided exploration. Use it when readers are meant to
manipulate or observe something dynamic rather than just read static
exposition.

A demonstration often includes a [`<title>`](./title.mdx), brief
explanatory [`<p>`](./p.mdx) content, and links or embedded interactive
material placed inside the block.

## Examples

```ptx-example
<FRAGMENT parents="section subsection">
    <demonstration>
        <title>Exploring slope with a draggable point</title>
        <p>Move the point along the curve and watch the secant line
        approach the tangent line.</p>
    </demonstration>
</FRAGMENT>
```
