# `<dl>`

The `<dl>` tag renders a list which has short pieces of text for their markers.

## Syntax

<SyntaxDisplay name="dl" />

## Examples

### An `<li>` of a description list must have a `<title>` element.

The title provides the text of the marker (rendered in boldface); the remainder of the list item must also be structured, typically with some paragraphs.  If the default format seems too constrained, perhaps a [`sidebyside`](./sidebyside) element might provide more flexibility.


 ```ptx-example
<FRAGMENT parents="p">
<dl>
    <li>
        <title>Groups of order 4:</title>
        <p>
        <m>\mathbb Z_2 \times \mathbb Z_2</m>, <m>\mathbb Z_4</m>
        </p>
    </li>
    <li>
        <title>Groups of order 5:</title>
        <p>
        <m>\mathbb Z_5</m>
        </p>
    </li>
    <li>
        <title>Groups of order 6:</title>
        <p>
        <m>\mathbb Z_6</m>, <m>S_3</m>
        </p>
    </li>
</dl>
</FRAGMENT>
```
