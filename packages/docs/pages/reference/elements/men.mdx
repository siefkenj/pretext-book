# `<men>`

Produces a numbered equation. Typically you will get vertical separation above and below, and the contents will be centered. If the equation concludes a sentence, put the concluding punctuation after the closing tag.

## Syntax

<SyntaxDisplay name="men" />

## Examples

```ptx-example
<FRAGMENT parents="p">
    The quadratic formula is <men>x=\frac{-b\pm\sqrt{b^2-4ac}}{2a}</men>.
</FRAGMENT>
```
