# `<instruction>`

The `<instruction>` tag gives short directions to the reader, usually
inside the [`<statement>`](./statement.mdx) of an interactive or WeBWorK
exercise. Use it for answer-entry guidance that should stand apart from
the main prompt.

## Syntax

<SyntaxDisplay name="instruction" />

## Examples

```ptx-example
<FRAGMENT parents="exercise section subsection statement">
    <instruction>
        Type your answer without using the <c>+</c> sign.
    </instruction>
</FRAGMENT>
```

```ptx-example
<FRAGMENT parents="exercise section subsection statement">
    <instruction>
        To enter a power, use <c>^</c>, as in <c>x^2</c> for <m>x^2</m>.
    </instruction>
</FRAGMENT>
```
