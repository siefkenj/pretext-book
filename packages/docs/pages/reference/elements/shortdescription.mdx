# `<shortdescription>`

The `<shortdescription>` element gives a brief plain-text summary of a
[`<figure>`](./figure.mdx) or [`<image>`](./image.mdx). It is useful for
metadata and accessibility situations where a compact description is
needed.

Unlike [`<description>`](./description.mdx), `<shortdescription>` should
stay short and unstructured.

<SyntaxDisplay name="shortdescription" />

## Examples

```ptx-example
<FRAGMENT parents="section subsection figure">
    <image source="graphs/parabola.png" width="50%">
        <shortdescription>
            A parabola opening downward with its vertex at the point
            (1, 3).
        </shortdescription>
    </image>
</FRAGMENT>
```

```ptx-example
<FRAGMENT parents="section subsection figure">
    <image source="diagrams/network.svg" width="70%">
        <shortdescription>
            Three computers connected to a central server.
        </shortdescription>
    </image>
</FRAGMENT>
```
