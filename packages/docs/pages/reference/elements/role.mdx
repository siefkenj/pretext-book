# `<role>`

The `<role>` tag names a person's contribution inside a
[`<credit>`](./credit.mdx) entry. It pairs with [`<entity>`](./entity.mdx)
to record work such as editing, design, or production.

## Syntax

<SyntaxDisplay name="role" />

## Examples

```ptx-example
<FRAGMENT parents="book frontmatter titlepage">
    <credit>
        <role>Cover Design</role>
        <entity>Covers 4 U</entity>
    </credit>
</FRAGMENT>
```
