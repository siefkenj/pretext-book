# `<articletitle>`

The `<articletitle>` element marks the title of a shorter work, such as an
article, essay, chapter, or webpage. It is the companion to
[`<pubtitle>`](./pubtitle.mdx), which is for larger stand-alone works.

## Syntax

<SyntaxDisplay name="articletitle" />

## Examples

```ptx-example
<FRAGMENT parents="p section subsection">
    We recommend
    <articletitle>On Computable Numbers</articletitle>
    as a starting point.
</FRAGMENT>
```

```ptx-example
<FRAGMENT parents="book backmatter references">

        <year>1937</year>.
    </biblio>
</FRAGMENT>
```
