# `<fillin>`

The self-closing `<fillin/>` tag is a generic fill-in-the-blank tag 
which can produce answer blanks as underline, boxes or shade.

## Syntax

<SyntaxDisplay name="fillin" />

### Remark
`<fillin>` has publication options which govern the style 
of fillin in both text and math modes.  
 Defaults are shown in the examples.

## Examples

### A `<fillin/>` in math

```ptx-example

<FRAGMENT parents="p">
    <c>&lt;m&gt;x^a * x^b = x^&lt;fillin fill="a+b" /&gt;&lt;/m&gt;</c> produces
    <m>x^a * x^b = x^<fillin fill="a+b" /></m>      
</FRAGMENT>

<FRAGMENT parents="p">
 <c>&lt;m&gt;x^a * x^b = x^&lt;fillin characters="3" /&gt;&lt;/m&gt;</c> produces
  <m>x^a * x^b = x^<fillin characters="3" /></m>   
</FRAGMENT>

```

### A `<fillin/>` in text

```ptx-example
<FRAGMENT parents="p">
 What is the ultimate answer to life, the universe, and everything?
 <fillin characters="5" />  (<c>&lt;fillin characters="5"/&gt;</c>)
</FRAGMENT>

<FRAGMENT parents="p">
 c>&lt;fillin rows="2" /&gt;</c> produces <fillin rows="2" />
</FRAGMENT>

<FRAGMENT parents="p">
 <c>&lt;fillin rows="4" cols="7" /&gt;</c> produces
<fillin rows="4" cols="7" />.</p>
</FRAGMENT>
```
