# `<stdin>`

The `<stdin>` element supplies standard input for a
[`<program>`](./program.mdx) example. It is useful when the program or
interactive code expects the user to type data while it runs.

Use `<stdin>` for the input stream itself; use [`<tests>`](./tests.mdx)
for automated checking of program behavior.
