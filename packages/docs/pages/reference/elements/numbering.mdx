# `<numbering>`

The `<numbering>` tag groups configuration for how items in a document
are numbered. It is commonly used in [`<docinfo>`](./docinfo.mdx), and
publisher-side numbering controls may also appear in a publication XML
file.

One common use is choosing whether [`<part>`](./part.mdx) divisions are
decorative or structural, which affects chapter numbering.
