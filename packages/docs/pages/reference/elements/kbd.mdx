# `<kbd>`

The `<kbd>` tag produces stylized text made to look like a key on a keyboard or calculator.

## Syntax

<SyntaxDisplay name="kbd" />

## Examples

### For keys labeled by text

```ptx-example
<FRAGMENT parents="p">
<c>&lt;kbd&gt;Alt&lt;/kbd&gt;+&lt;kbd&gt;Z&lt;/kbd&gt;</c>
  produces <kbd>Alt</kbd>+<kbd>Z</kbd>.
</FRAGMENT>
```

### For keys labeled by symbols

```ptx-example
<FRAGMENT parents="p">
 <c>&lt;kbd name="inverse" /&gt; &lt;kbd name ="enter" /&gt;</c>
  produces <kbd name="inverse" /> <kbd name="enter" />.
</FRAGMENT>
```
