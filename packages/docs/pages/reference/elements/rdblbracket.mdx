# `<rdblbracket>`

The empty `<rdblbracket/>` tag inserts the right double bracket symbol ⟧. It
is the closing partner of [`<ldblbracket/>`](./ldblbracket.mdx) for inline
notation.

## Syntax

<SyntaxDisplay name="rdblbracket" />

## Examples

```ptx-example
<FRAGMENT parents="p section subsection">
    The Iverson-style expression is <ldblbracket/>P<rdblbracket/>.
</FRAGMENT>
```

```ptx-example
<FRAGMENT parents="p section subsection">
    Write <ldblbracket/>n is even<rdblbracket/> for the indicator value.
</FRAGMENT>
```
