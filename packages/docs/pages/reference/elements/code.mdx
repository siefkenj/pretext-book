# `<code>`

The `<code>` tag contains the verbatim source text of a
[`<program>`](./program.mdx). It is especially useful when a program
also has siblings such as [`<preamble>`](./preamble.mdx) or
[`<tests>`](./tests.mdx), or when the source is kept in a separate file.

If a `<program>` contains only code, the text may appear directly in the
program, but `<code>` makes the source block explicit.
