# `<tag>`

Used the wrap the name of a tag/element when writing about PreTeXt 
or other XML, as in, `The <tag>image</tag> element is used to...` 

See also [`<tage>`](./tage) and [`<attr>`](./attr) for other markup-describing
elements.
