# `<commentary>`

The `<commentary>` tag is an aside-like block for author notes,
editorial remarks, or other commentary about nearby content. It is
similar in spirit to [`<aside>`](./aside.mdx) or
[`<remark>`](./remark.mdx), but is often intended for commentary that
may be hidden from ordinary readers.

Use it when you want material for instructors, editors, or specialized
audiences kept distinct from the main exposition.
