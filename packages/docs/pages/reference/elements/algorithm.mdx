# `<algorithm>`

`<algorithm>` is a [`<theorem>`](theorem)-like tag
and so shares the attributes and structure with other theorem-like tags.
In particular, the `<algorithm>` tag creates a container for the
statement of an algorithm with optional [`title`](./title),
[`creator`](./creator), and [`proof`](./proof).

## Examples

            <li><p>Repeat.</p></li>
        </ol>
    </statement>
</algorithm>
