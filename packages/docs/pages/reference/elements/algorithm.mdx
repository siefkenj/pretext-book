# `<algorithm>`

The `<algorithm>` tag creates a container for the statement of an algorithm
with optional `title`, `creator`, and `proof`. The structure of the `algorithm`
tag is completely analogous to that of the following tags:
[`claim`](./claim), [`corollary`](./corollary),
[`fact`](./fact), [`identity`](./identity), [`lemma`](./lemma),
[`proposition`](./proposition), and [`theorem`](./theorem).

## Examples

 ```ptx-example
<FRAGMENT parents="p">
<algorithm xml:id="alg-infinite-loop">
    <title>Instructions on the shampoo bottle</title>
    <statement>
        <ol>
            <li><p>Lather.</p></li>
            <li><p>Rinse.</p></li>
            <li><p>Repeat.</p></li>
        </ol>
    </statement>
</algorithm>
</FRAGMENT>
```
