# `<subtitle>`

The `<subtitle>` tag can be used only in the narrow context to provide a
subtitle to a book's or ariticle's main title.

## Syntax

<SyntaxDisplay name="subtitle" />

## Examples

```ptx-block
<FRAGMENT>
<article>
    <title>Article Title</title>
    <subtitle>Perhaps a subtitle</subtitle>
    ...
</article>
</FRAGMENT>
```
