# `<genus>`

The `<genus>` tag gives the genus portion of a scientific name inside a
[`<taxon>`](./taxon.mdx). It is usually paired with [`<species>`](./species.mdx)
and other taxonomic parts to format organism names consistently.

Use `<genus>` when marking up biological classification rather than ordinary
italicized text.

## Syntax

<SyntaxDisplay name="genus" />

## Examples

```ptx-example
<FRAGMENT parents="p section subsection">
    The gray wolf belongs to the taxon <taxon><genus>Canis</genus>
    <species>lupus</species></taxon>.
</FRAGMENT>
```

```ptx-example
<FRAGMENT parents="p section subsection">
```
