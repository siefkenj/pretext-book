# `<output>`

The `<output>` tag records the text produced by a command inside a
[`<console>`](./console.mdx) transcript. Pair it with
[`<input>`](./input.mdx) to show what a user typed and what the system
returned.

Use one `<output>` element for each chunk of terminal output you want to
display.

## Examples

```ptx-example
<FRAGMENT parents="section subsection">
    <console prompt="$ ">
        <input>python hello.py</input>
        <output>Hello, world!</output>
    </console>
</FRAGMENT>
```
