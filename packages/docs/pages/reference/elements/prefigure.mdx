# `<prefigure>`

The empty `<prefigure/>` tag inserts the name of the PreFigure diagram system
inline. Use it in prose when referring to PreFigure as a tool, much like
[`<pretext/>`](./pretext.mdx) names PreTeXt.

This tag is not the external PreFigure diagram markup itself. Instead, it is
a convenient inline reference to the PreFigure project.
