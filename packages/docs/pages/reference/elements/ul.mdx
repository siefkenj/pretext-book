# `<ul>`

The `<ul>` tag renders an unordered list. E.g., a list with bullets/dashes/some other unordered marker. For comparison, see [`<ol>`](./ol)

## Syntax

<SyntaxDisplay name="ul" />

## Examples

```ptx-example
<FRAGMENT parents="p">
    <ul>
        <li>Item 1</li>
        <li>Item 2</li>
    </ul>
</FRAGMENT>
```

```ptx-example
<FRAGMENT parents="p">
    <ul marker="square">
        <li>Item 1</li>
        <li>Item 2</li>
    </ul>
</FRAGMENT>
```

```ptx-example
<FRAGMENT parents="p">
    <ul cols="3">
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
        <li>Item 4</li>
    </ul>
</FRAGMENT>
```
