# `<see>`

The `<see>` tag is used to create a cross-reference entry for an item in the
Index.  It is functionally equivalent to the [`<seealso>`](./seealso) tag.

## Syntax

<SyntaxDisplay name="see" />

## Examples

Assuming the Index page contains a list of notation, and the notation <m>\trianglelefteq</m> has been introduced to indicate a
normal subgroup, the following `<see>` tag would create an entry next to that
symbol with the cross-reference to see `normal subgroup` for further information.

```ptx-example
<FRAGMENT>
<p>
    <idx><h>notation</h><h><m>\trianglelefteq</m></h><see>normal subgroup</see></idx>
</p>

<backmatter>
 <title>Backmatter</title>
      <index>
        <title>Index</title>
        <index-list/>
      </index>
</backmatter>
</FRAGMENT>
```
