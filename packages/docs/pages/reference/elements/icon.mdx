# `<icon>`

The self-closing `<icon/>` tag produces icons known to Font Awesome and which have a unicode representation.

## Syntax

<SyntaxDisplay name="icon" />

## Examples

```ptx-example
<FRAGMENT parents="p">
     Icons for menu, power, and media-play:
     <icon name="menu"/>, <icon name="power"/>, <icon name="media-play"/>.
</FRAGMENT>
```
