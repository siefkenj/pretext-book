# `<author-biographies>`

The empty `<author-biographies>` element is a
[`<docinfo>`](./docinfo.mdx) setting that tells PreTeXt to include author
[`<biography>`](./biography.mdx) entries in the output.

Its `length` attribute chooses whether short or long biographies are shown.

## Syntax

<SyntaxDisplay name="author-biographies" />

## Examples

```ptx-example
<FRAGMENT parents="book docinfo">
    <author-biographies length="short" />
</FRAGMENT>
```

```ptx-example
<FRAGMENT parents="book docinfo">
    <author-biographies length="long" />
</FRAGMENT>
```
