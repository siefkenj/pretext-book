# `<abbr>`

Wrapper for an abbreviation inside a paragraph or other block of text.  
For example, `Many people write <abbr>Blvd.</abbr> instead of Boulevard.`

Specifically for abbreviations that are neither initialisms (see [`<init>`](./init)) 
nor acronyms (see [`<acro>`](./acro)).

## Syntax

<SyntaxDisplay name="abbr" />

## Examples

 ```ptx-example
<FRAGMENT parents="p">
    <abbr>Corp.</abbr> is an abbreviation, while <init>NSA</init> is an
    initialism and <acro>LASIK</acro> is an acronym (which are both types
    of abbreviations).
</FRAGMENT>
```

 ```ptx-example
<FRAGMENT parents="p">
    Abbreviations can be shortened words, like <abbr>Prof.</abbr> or 
    <abbr>Ms.</abbr> that are still read as the original word.
</FRAGMENT>
```
