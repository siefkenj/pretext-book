# `<article>`

The `<article>` element is a root-level document type for a shorter work
than a [`<book>`](./book.mdx). An article can have
[`<frontmatter>`](./frontmatter.mdx), sections, and
[`<backmatter>`](./backmatter.mdx), but it does not contain chapters.

## Examples

```ptx-example
<article>
    <title>Notes on Counting</title>
    <frontmatter>
        <bibinfo>
            <author><personname>Emmy Noether</personname></author>
        </bibinfo>
        <titlepage><titlepage-items /></titlepage>
    </frontmatter>
    <section>
        <title>Introduction</title>
        <p>We begin with a simple counting argument.</p>
    </section>
</article>
```
