# `<contributor>`

The `<contributor>` tag records one person in a
[`<contributors>`](./contributors.mdx) list. It normally contains a
[`<personname>`](./personname.mdx) and may include affiliation details
such as [`<department>`](./department.mdx),
[`<institution>`](./institution.mdx), [`<location>`](./location.mdx),
or [`<email>`](./email.mdx).

Use one `<contributor>` per individual you want to acknowledge.

## Examples

```ptx-example
<FRAGMENT parents="frontmatter preface contributors">
    <contributor>
        <personname>Jordan Lee</personname>
    </contributor>
</FRAGMENT>
```

```ptx-example

        <email>priya@example.edu</email>
    </contributor>
</FRAGMENT>
```
