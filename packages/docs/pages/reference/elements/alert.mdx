# `<alert>`

Wrapper for a string of text that is heavily emphasized, to a greater degree than just emphasis (see [`<em>`](./em)).

## Syntax

<SyntaxDisplay name="alert" />

## Examples

 ```ptx-example
<FRAGMENT parents="p">
    In case of fire, <alert>don't panic</alert>!  
</FRAGMENT>
```

 ```ptx-example
<FRAGMENT parents="p">
    You should <alert>never</alert> shout, but you <em>could</em>
    raise your voice slightly.
</FRAGMENT>
```
