# `<iotest>`

The `<iotest>` tag records one input/output test case inside
[`<tests>`](./tests.mdx) for a [`<program>`](./program.mdx) or similar
coding example. Each case contains one [`<input>`](./input.mdx) and the
expected [`<output>`](./output.mdx).

        <output>*</output>
    </iotest>
</FRAGMENT>
