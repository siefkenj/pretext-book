# `<timeofday>`

The `<timeofday>` element inserts the current time when the document is
processed. It is often paired with [`<today>`](./today.mdx) for build or
revision timestamps.

Because the value is generated dynamically, `<timeofday>` is useful for
draft notices and generated-output metadata.
