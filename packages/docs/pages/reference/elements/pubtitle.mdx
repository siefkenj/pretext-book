# `<pubtitle>`

The `<pubtitle>` tag marks the title of a published work in running prose. It
is an inline element, typically rendered in italics, for book, journal, or
collection titles.

Use [`<pubtitle>`](./pubtitle.mdx) for the container publication, not for the
title of an article or chapter; compare [`<articletitle>`](./articletitle.mdx).
