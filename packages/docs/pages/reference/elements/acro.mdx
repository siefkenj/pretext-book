# `<acro>`

Use this tag for an acronym, which is an abbreviation consisting of a sequence of letters, where the letters are read as a pronounceable word. If the abbreviation should instead be read as a sequence of letters, see <init>, initialism.

## Syntax

### Attributes

<AttrDisplay name="acro" />

### Children

The following elements can be used as children.

<ChildrenDisplay name="acro" />

### Child of

This element can appear as an immediate child of the following elements.

<ParentsDisplay name="acro" />

## Examples

<FRAGMENT parents="p">
    <acro>SCUBA</acro> is an acronym which stands for <q>Self-Contained Underwater Breathing Apparatus.</q> 
</FRAGMENT>
