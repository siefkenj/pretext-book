# `<acro>`

Use this tag for an acronym, which is an abbreviation consisting of a sequence of letters,
where the letters are read as a pronounceable word. If the abbreviation should instead be read as a sequence of letters, see [`<init>`](init), initialism.

## Syntax

<SyntaxDisplay name="acro" />

## Examples

```ptx-example
<FRAGMENT parents="p">
    <acro>SCUBA</acro> is an acronym which stands for
    <q>Self-Contained Underwater Breathing Apparatus.</q> 
</FRAGMENT>
```
