# `<holder>`

The `<holder>` tag gives the name of the copyright holder inside a
[`<copyright>`](./copyright.mdx) statement. It identifies the person,
organization, or publisher that owns the copyright.

Use `<holder>` together with elements such as `<year>` to build a complete
copyright notice.

<SyntaxDisplay name="holder" />

## Examples

```ptx-example
<FRAGMENT parents="frontmatter bibinfo copyright">
    <holder>Jane Smith</holder>
</FRAGMENT>
```
