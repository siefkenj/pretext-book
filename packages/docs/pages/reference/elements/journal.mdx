# `<journal>`

The `<journal>` tag gives the journal name for an article cited in a
[`<biblio>`](./biblio.mdx) entry. It commonly appears with
[`<volume>`](./volume.mdx), [`<number>`](./number.mdx), and
[`<year>`](./year.mdx).

## Examples

```ptx-example
<FRAGMENT parents="section references biblio">

</FRAGMENT>
```
