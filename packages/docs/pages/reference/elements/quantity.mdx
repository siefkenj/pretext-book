# `<quantity>`

The `<quantity>` tag marks up a physical quantity in prose. It usually
contains a magnitude in [`<mag>`](./mag.mdx), followed by one or more
[`<unit>`](./unit.mdx) elements, optionally divided by units in
[`<per>`](./per.mdx).

Use `<quantity>` instead of plain text when you want units formatted
consistently and semantically.

<SyntaxDisplay name="quantity" />

## Examples

```ptx-example
<FRAGMENT parents="p section subsection">
    The car was traveling at
    <quantity><mag>5</mag><unit>m</unit><per>s</per></quantity>.
</FRAGMENT>
```

```ptx-example
<FRAGMENT parents="p section subsection">
    The board is <quantity><mag>2.4</mag><unit>m</unit></quantity> long.
</FRAGMENT>
```
