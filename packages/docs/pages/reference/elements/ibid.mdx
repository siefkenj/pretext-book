# `<ibid>`

The `<ibid>` tag is an inline bibliographic marker meaning "the same source as
the previous citation." It is typically used inside a raw [`<biblio>`](./biblio.mdx)
entry when successive references repeat the same work.

Because it depends on the preceding entry, `<ibid>` only makes sense in a
bibliographic context.
