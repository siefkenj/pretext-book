# `<n>`

The self-closing `<n/>` tag displays detailed symbol about a musical note.
## Syntax

<SyntaxDisplay name="n" />

## Examples

 
```ptx-example
<FRAGMENT parents="p">
  Middle C is the fourth C on the 88-key piano keyboard denoted 
  <n pc="C" octave="4"/>. One denotes a C<sharp/> in the same octave as 
  <n pc="C" acc="sharp" octave="4" />, written in PreTeXt as 
  <c>&lt;n pc="C" acc="sharp" octave="4"/&gt;</c>.
</FRAGMENT>
```
