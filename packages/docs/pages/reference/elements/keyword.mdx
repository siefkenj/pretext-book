# `<keyword>`

The `<keyword>` tag supplies one subject term inside a
[`<keywords>`](./keywords.mdx) list. A keyword can also be marked as the
primary entry with the <attr>primary</attr> attribute.

## Syntax

<SyntaxDisplay name="keyword" />

## Examples

```ptx-example
<FRAGMENT parents="pretext docinfo keywords">
    <keyword>graph theory</keyword>
</FRAGMENT>
```

```ptx-example
<FRAGMENT parents="pretext docinfo keywords">
    <keyword primary="yes">05C10</keyword>
</FRAGMENT>
```
