# `<abstract>`

The `<abstract>` tag encloses an article's abstract.  This is only valid in an `<article>` (not a `<book>`).

## Syntax

<SyntaxDisplay name="abstract" />

## Examples

```ptx-example
<FRAGMENT parents="frontmatter">
    <abstract>
        <p>
            In this article we will demonstrate how to use an abstract.
        </p>
    </abstract>
</FRAGMENT>
```
