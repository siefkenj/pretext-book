# `<appendix>`

The `<appendix>` element is a major division in the
[`<backmatter>`](./backmatter.mdx) of a [`<book>`](./book.mdx) or
[`<article>`](./article.mdx). It behaves much like a chapter or section,
but is labeled as an appendix.

## Syntax

<SyntaxDisplay name="appendix" />

## Examples

```ptx-example
<FRAGMENT parents="book backmatter">
    <appendix>
        <title>Notation</title>
        <p>
            This appendix collects the symbols used throughout the text.
        </p>
    </appendix>
</FRAGMENT>
```
