# `<index-list>`

The empty `<index-list>` tag appears inside an
[`<index>`](./index.mdx) division and tells PreTeXt to generate the
finished index automatically from all [`<idx>`](./idx.mdx) entries in
the document.

## Syntax

<SyntaxDisplay name="index-list" />

## Examples

```ptx-example
<FRAGMENT parents="book backmatter index">
    <index-list />
</FRAGMENT>
```

```ptx-example
<FRAGMENT parents="book backmatter">
    <index>
        <title>Index</title>
        <index-list />
    </index>
</FRAGMENT>
```
