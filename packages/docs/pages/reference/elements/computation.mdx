# `<computation>`

The `<computation>` tag is a computation-like block for examples
or activities centered on calculation, software, or algorithmic work.
It shares the same general role as [`<technology>`](./technology.mdx)
and [`<data>`](./data.mdx), but emphasizes the computational process
itself.

A `<computation>` can mix explanatory prose with things like
[`<program>`](./program.mdx) or data resources.

## Examples

```ptx-example
<FRAGMENT parents="section subsection">
    <computation>
        <title>Estimating a derivative numerically</title>
        <p>Use a small value of <m>h</m> and compare several difference
        quotients.</p>
    </computation>
</FRAGMENT>
```

```ptx-example
<FRAGMENT parents="section subsection">
    <computation>
        <title>Summing the first ten squares</title>

            ]]></code>
        </program>
    </computation>
</FRAGMENT>
```
