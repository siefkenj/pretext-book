# `<colophon-items>`

The `<colophon-items>` element is an empty helper used inside a
front [`<colophon>`](./colophon.mdx). It tells PreTeXt to collect the
appropriate publication details automatically from
[`<bibinfo>`](./bibinfo.mdx).

Use `<colophon-items />` instead of manually repeating copyright-page
metadata inside the colophon.

## Examples

```ptx-example
<FRAGMENT parents="frontmatter colophon">
    <colophon-items />
</FRAGMENT>
```

```ptx-example
<FRAGMENT parents="frontmatter">

    </colophon>
```
