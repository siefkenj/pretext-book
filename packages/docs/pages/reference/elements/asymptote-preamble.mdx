# `<asymptote-preamble>`

The `<asymptote-preamble>` element stores Asymptote code that should be
added before every [`<asymptote>`](./asymptote.mdx) graphic in the
document. Place it in [`<docinfo>`](./docinfo.mdx) when several images
share imports, size settings, or helper definitions.

## Syntax

<SyntaxDisplay name="asymptote-preamble" />

## Examples

```ptx-example
<FRAGMENT parents="article docinfo">
    <asymptote-preamble>
import graph;
defaultpen(linewidth(0.8));
    </asymptote-preamble>
</FRAGMENT>
```
