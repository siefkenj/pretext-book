# `<taxon>`

The `<taxon>` element marks up a scientific name using taxonomic parts
such as [`<genus>`](./genus.mdx) and [`<species>`](./species.mdx). It
provides semantic markup for biological names rather than plain
formatting.

Use `<taxon>` when you want PreTeXt to treat a scientific name as a
classification term.

<SyntaxDisplay name="taxon" />

## Examples

```ptx-example
<FRAGMENT parents="p section subsection">
    The fruit fly is
    <taxon><genus>Drosophila</genus> <species>melanogaster</species></taxon>.
</FRAGMENT>
```

