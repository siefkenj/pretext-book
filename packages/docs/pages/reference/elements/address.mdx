# `<address>`

The `<address>` element gives the address portion of a
[`<website>`](./website.mdx) entry, usually in a
[`<colophon>`](./colophon.mdx) or other front matter metadata.

## Syntax

<SyntaxDisplay name="address" />

## Examples

```ptx-example
<FRAGMENT parents="frontmatter colophon">
    <website>
        <name>PreTeXt Project</name>
        <address>https://pretextbook.org</address>
    </website>
</FRAGMENT>
```
