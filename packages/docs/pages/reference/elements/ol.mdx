# `<ol>`

The `<ol>` tag renders an ordered list with default markers which can (if
desired) be modified.  For comparison, see [`<ul>`](./ul).

## Syntax

<SyntaxDisplay
    name="ol"
    defaultVariant="Ol"
    additionalAttributeInfo={{
        Ol: { marker: { value: "(1) (A) (a) (I) (i)" } }
    }}
/>

## Examples

### A sample nested ordered list with default markers.

 ```ptx-example
<FRAGMENT parents="p">
<ol>
    <li>Call the meeting to order</li>
    <li>Approval of the minutes</li>
    <li>
        <p>New Business
        <ol>
            <li>Approve new course</li>
            <li>
            <p>Vote on honors theses:
            <ol>
                <li>C. F. Gauss</li>
                <li>Euclid</li>
                <li>E. Noether</li>
            </ol>
            </p>
            </li>
        </ol>
        </p>
    </li>
    <li>Old Business</li>
</ol>
</FRAGMENT>
```

### An ordered list with custom markers.

 ```ptx-example
<FRAGMENT parents="p">
<ol marker="(1)">
    <li>Call the meeting to order</li>
    <li>Approval of the minutes</li>
</ol>
</FRAGMENT>
```

### To reference a list, place it inside a [`<list>`](list) environment.

 ```ptx-example
<FRAGMENT parents="p">
<list>
    <title>Meeting Agenda</title>
    <ol marker="(1)">
        <li>Call the meeting to order</li>
        <li>Approval of the minutes</li>
    </ol>
</list>
</FRAGMENT>
```
