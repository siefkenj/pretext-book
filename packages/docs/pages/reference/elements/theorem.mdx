# `<theorem>`

The `<theorem>` tag creates a container for the statement of a theorem with
optional  [`title`](./title), [`creator`](./creator), and [`proof`](./proof).
Other tags which have the same attributes and structure as the `<theorem`>
tag include:
[`algorithm`](.algorithm), [`claim`](./claim), [`corollary`](./corollary),
[`fact`](./fact), [`identity`](./identity), [`lemma`](./lemma),
and [`proposition`](./proposition).

<SyntaxDisplay name="theorem" />

## Examples

 ```ptx-example
<FRAGMENT parents="section subsection">
<theorem>
    <title>Optional title</title>
    <creator>Lagrange</creator>
    <statement>
        <p>
        Let <m>G</m> be a finite group and <m>H</m> a subgroup.
        Then the order of <m>H</m> divides the order of <m>G</m>.
        </p>
    </statement>
    <proof>
        <p>
        The left cosets of <m>H</m> partition the group,
        and all have the same size.
        </p>
    </proof>
</theorem>
</FRAGMENT>
```
