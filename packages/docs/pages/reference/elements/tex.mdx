# `<tex>`

The `<tex>` element inserts the stylized TeX logo in running text. It
is an inline convenience element, similar to [`<latex>`](./latex.mdx)
for the LaTeX logo.

Use `<tex/>` when referring to the TeX typesetting system by name.
