# `<division>`

The `<division>` tag is a configuration element used inside
[`<numbering>`](./numbering.mdx) within [`<docinfo>`](./docinfo.mdx). It
controls whether division numbers use their structural form or their
decorative form.

This is not a content division like [`<chapter>`](./chapter.mdx) or
[`<section>`](./section.mdx); it only affects numbering configuration.
