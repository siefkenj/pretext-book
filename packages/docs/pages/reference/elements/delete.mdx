# `<delete>`

The `<delete>` tag marks text that has been removed as part of an
edit or revision. It is typically used alongside
[`<insert>`](./insert.mdx) and [`<stale>`](./stale.mdx) when you want to
show changes explicitly in running text.

In output, deleted text is usually styled so readers can distinguish it
from the current wording.
