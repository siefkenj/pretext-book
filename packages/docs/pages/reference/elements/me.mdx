# `<me>`

Expressions enclosed by the `<me>` tag are rendered as a single line of displayed mathematics.
By contrast, the [`<m>`](./m) tag is used for in-line mathematics.

Note: If an equation concludes a sentence, put the concluding punctuation after the closing tag.

<SyntaxDisplay name="me" />

## Examples

```ptx-example
<FRAGMENT parents="p">
    <me>\int \cos x\, dx=\sin x+C</me>
</FRAGMENT>
```
