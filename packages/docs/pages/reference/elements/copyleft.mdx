# `<copyleft>`

Produces the copyleft symbol, which is like copyright, but backwards.
The unicode character is &#x1f12f;, which is entered as `&#x1f12f;` in HTML.
Used with the self-closing syntax `<copyleft/>`.

## Syntax

### Attributes

<AttrDisplay name="copyleft" />

### Children

The following can be used as children.

<ChildrenDisplay name="copyleft" />

### Child of

This element can appear as an immediate child of the following elements.

<ParentsDisplay name="copyleft" />

## Examples
```ptx-eample
<FRAGMENT parents="p">
  copyleft <copyleft/> 1993, all rights reserved.
</FRAGMENT>
```
