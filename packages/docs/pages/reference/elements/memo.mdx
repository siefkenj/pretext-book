# `<memo>`

The `<memo>` tag is a root-level document type for a memorandum. Like
[`<letter>`](./letter.mdx) or [`<article>`](./article.mdx), it starts a
document, but it is tailored to internal correspondence with memo-style
front matter such as to, from, subject, and date.

A memo can still contain ordinary PreTeXt body content, including
paragraphs and divisions like [`<paragraphs>`](./paragraphs.mdx).

## Examples

```ptx-example
<FRAGMENT parents="pretext">
    <memo xml:id="team-memo">
        <title>Department Memo</title>
        <frontmatter>
            <to>Faculty</to>
            <from>Department Chair</from>
            <subject>Schedule Update</subject>
            <date>2025-08-15</date>
        </frontmatter>
        <p>The meeting has moved to Friday afternoon.</p>
    </memo>
</FRAGMENT>
```
