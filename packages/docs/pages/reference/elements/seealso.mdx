# `<seealso>`

The `<seealso>` tag is used to create a cross-reference entry for an item in the
Index.  It is functionally equivalent to the [`<see>`](./see) tag.

## Syntax

<SyntaxDisplay name="seealso" />

## Examples

In abstract algebra, the term `normal` could refer to normal subgroups or
to normal field extensions.  One might choose to create a cross-reference
from normal field extensions to Galois extensions as a point of further
investigaton.

```ptx-example
<FRAGMENT>
<p>
    <idx><h><normal</h><h>field extension</h><seealso>Galois</seealso></idx>
</p>

<backmatter>
 <title>Backmatter</title>
      <index>
        <title>Index</title>
        <index-list/>
      </index>
</backmatter>
```
