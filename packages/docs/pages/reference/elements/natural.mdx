# `<natural>`

The self-closing `<natural/>` tag displays the natural symbol in music notation. 

## Syntax

<SyntaxDisplay name="natural" />

## Examples

```ptx-example
<FRAGMENT parents="p">
<p>
    In the context of music, the natural symbol (<natural/>) cancels a flat or
    sharp for a key signature or note, so A<natural/><sharp/> is an A.
</p>
</FRAGMENT>
```
