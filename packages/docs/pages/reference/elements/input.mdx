# `<input>`

The `<input>` tag records text entered by a reader or user. It is most
commonly used inside [`<console>`](./console.mdx) to show typed commands,
and it can pair with [`<output>`](./output.mdx) or appear inside
[`<iotest>`](./iotest.mdx) test cases.
