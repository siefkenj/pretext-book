# `<keywords>`

The `<keywords>` tag belongs in [`<docinfo>`](./docinfo.mdx) and groups
subject terms or classification codes for the document. It contains one
or more [`<keyword>`](./keyword.mdx) elements and may use attributes such
as <attr>authority</attr> and <attr>variant</attr>.

<SyntaxDisplay name="keywords" />

## Examples

```ptx-example
<FRAGMENT parents="pretext docinfo">
    <keywords>
        <keyword>linear algebra</keyword>
        <keyword>eigenvalues</keyword>
    </keywords>
</FRAGMENT>
```

```ptx-example
<FRAGMENT parents="pretext docinfo">
    <keywords authority="ams" variant="msc">

    </keywords>
</FRAGMENT>
```
