# `<frontmatter>`

The `<frontmatter>` tag is a wrapper for material that comes before the first [`<chapter>`](./chapter) (or [`<part>`](./part)) or [`<section>`](./section) of a [`<book>`](./book) or [`<article>`](./article).  It contains author information, as well as other bibliographic information (contained in the [`<bibinfo>`](./bibinfo)), and perhaps a number [`<preface>`](./preface) tags or an [`<abstract>`](./abstract).  Note that the [`<title>`](./title) of the document does NOT go inside `<frontmatter>`.
