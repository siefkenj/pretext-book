# `<cline>`

The `<cline>` tag represents one line of verbatim text inside a
displayed code block. It is most commonly used as a child of
[`<cd>`](./cd.mdx) when you want each line break to be explicit.

Use separate `<cline>` elements instead of embedded newlines when exact
line structure matters.
