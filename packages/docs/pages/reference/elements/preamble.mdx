# `<preamble>`

In a [`<program>`](./program.mdx), the `<preamble>` tag holds text that comes
before the main [`<code>`](./code.mdx) block. It is useful for imports,
helper definitions, or setup code that belongs with the listing.

A `<program>` may use [`<preamble>`](./preamble.mdx),
[`<code>`](./code.mdx), and [`<postamble>`](./postamble.mdx) together to show
distinct parts of a program.

## Examples

```ptx-example
<FRAGMENT parents="section subsection">
    <program language="python">
        <preamble visible="yes">import math</preamble>

print(math.sqrt(49))
        ]]></code>
    </program>
</FRAGMENT>
```

}
