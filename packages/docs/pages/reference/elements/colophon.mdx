# `<colophon>`

The `<colophon>` tag contains publication details that appear in
front or back matter. In the front matter of a [`<book>`](./book.mdx),
it usually contains the empty
[`<colophon-items>`](./colophon-items.mdx) element so PreTeXt can
generate a copyright page from [`<bibinfo>`](./bibinfo.mdx).

A back-matter `<colophon>` can instead contain ordinary block content
such as [`<p>`](./p.mdx) with production notes or edition history.
