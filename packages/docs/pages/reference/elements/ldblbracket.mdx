# `<ldblbracket>`

The empty `<ldblbracket>` tag inserts the left double-bracket symbol
⟦ inline. It is usually paired with [`<rdblbracket>`](./rdblbracket.mdx)
when you need semantic brackets or interval notation without typing the
Unicode character directly.

## Syntax

<SyntaxDisplay name="ldblbracket" />

## Examples

```ptx-example
<FRAGMENT parents="p section subsection">
    The Iverson bracket is <ldblbracket /> P <rdblbracket />.
</FRAGMENT>
```

