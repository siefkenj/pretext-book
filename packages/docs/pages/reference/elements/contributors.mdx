# `<contributors>`

The `<contributors>` tag groups a list of individual
[`<contributor>`](./contributor.mdx) entries. It is useful in front or
back matter when you want to acknowledge people who contributed to the
project but are not listed as document authors.

Each contributor is recorded separately so names and affiliations can be
structured consistently.

## Syntax

<SyntaxDisplay name="contributors" />

## Examples

```ptx-example
<FRAGMENT parents="frontmatter preface">
    <contributors>
        <contributor>
            <personname>Jordan Lee</personname>
        </contributor>
        <contributor>
            <personname>Priya Shah</personname>
        </contributor>
    </contributors>
</FRAGMENT>
```

```ptx-example
<FRAGMENT parents="frontmatter preface">
    <contributors>
        <contributor>
            <personname>Alex Romero</personname>
            <institution>Open Learning Lab</institution>
        </contributor>
    </contributors>
</FRAGMENT>
```
