# `<lemma>`

`<lemma>` is a [`<theorem>`](theorem)-like tag
and so shares the attributes and structure with other theorem-like tags.
In particular, the `<lemma>` tag creates a container for the
statement of a lemma with optional [`title`](./title),
[`creator`](./creator), and [`proof`](./proof).

## Syntax

<SyntaxDisplay name="lemma" />

## Examples

```ptx-example
<FRAGMENT parents="section subsection">
Gauss's lemma is a key component used to show that
the ring of polynomials with integer coefficients,
<m>\mathbb Z[x]</m>, is a unique factorization domain.

<lemma>
  <title>Gauss's lemma for polynomials</title>
  <statement>
    <p>
     If <m>f,g \in \mathbb Z[x]</m> are primitive polynomials,
     then so is their product, <m>fg</m>.
    </p>
  </statement>
</lemma>
</FRAGMENT>
```
