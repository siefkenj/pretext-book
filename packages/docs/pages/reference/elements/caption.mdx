# `<caption>`

Contains the caption for a `figure` or `listing`.
It is a mandatory component of both elements. 

## Syntax

### Attributes

<AttrDisplay name="caption" />

### Children

The following can be used as children.

<ChildrenDisplay name="caption" />

### Child of

This element can appear as an immediate child of the following elements.

<ParentsDisplay name="caption" />
