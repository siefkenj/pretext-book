# `<bc>`

The self-closing `<bc/>` tag displays the abbreviation (BC)
 for the English phrase "before Christ" indicating dates before the Common Era (CE).

## Syntax

<SyntaxDisplay name="bc" />

## Examples

```ptx-example
<FRAGMENT parents="p">
    The year following 1 <bc/> was 1 <ad/>.
</FRAGMENT>
```
