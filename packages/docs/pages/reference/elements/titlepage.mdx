# `<titlepage>`

The `<titlepage>` element creates a document title page inside
[`<frontmatter>`](./frontmatter.mdx). It usually contains
[`<titlepage-items>`](./titlepage-items.mdx), which gathers the relevant
information automatically from [`<bibinfo>`](./bibinfo.mdx).

Use `<titlepage>` when you want a separate title page in a
[`<book>`](./book.mdx) or [`<article>`](./article.mdx).
