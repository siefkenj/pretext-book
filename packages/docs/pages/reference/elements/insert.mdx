# `<insert>`

The `<insert>` tag marks text that has been added as part of an
editorial revision. It is commonly used with [`<delete>`](./delete.mdx)
and [`<stale>`](./stale.mdx), often inside an
[`<alteration>`](./alteration.mdx) workflow.

    This sentence has <insert>new</insert> wording.
