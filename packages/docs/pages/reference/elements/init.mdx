# `<init>`

The `<init>` tag marks an inline initialism, an abbreviation read one
letter at a time, such as "HTML" or "NSA". Use it instead of
[`<abbr>`](./abbr.mdx) or [`<acro>`](./acro.mdx) when the letters are
spoken individually.
