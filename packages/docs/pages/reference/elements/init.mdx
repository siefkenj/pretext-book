# `<init>`

Use this tag for an initialism, which is an abbreviation read as a sequence of letters, often the first letter of words in a phrase. If the abbreviation should instead be read as a pronounceable word, see <acro>, acronym.

## Syntax

### Attributes

<AttrDisplay name="init" />

### Children

The following elements can be used as children.

<ChildrenDisplay name="init" />

### Child of

This element can appear as an immediate child of the following elements.

<ParentsDisplay name="init" />

## Examples

```ptx-example
<FRAGMENT parents="p">
    <init>HTML</init> is an abbreviation for <q>HyperText Markup Language.</q>
</FRAGMENT>
```
