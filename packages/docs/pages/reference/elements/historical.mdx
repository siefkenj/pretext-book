# `<historical>`

The `<historical>` tag is a remark-like block for historical context,
background, or anecdotes. It is useful when you want to separate historical
material from the main exposition.

Like [`<aside>`](./aside.mdx) or [`<biographical>`](./biographical.mdx), a
`<historical>` block can contain a title and one or more paragraphs.

## Syntax

<SyntaxDisplay name="historical" />

## Examples

```ptx-example
<FRAGMENT parents="section subsection">
    <historical>
        <title>Origins of the notation</title>
        <p>
            The symbol <m>\pi</m> was popularized in the eighteenth century,
            though earlier authors used other notations.
        </p>
    </historical>
</FRAGMENT>
```

