# `<tage>`

Used to wrap the name of an empty tag/element when writing about
PreTeXt or other XML, as in, `The <tage>latex</tage> element is self-closing.`

See also [`<attr>`](./attr) and [`<tag>`](./tag) for other markup-describing
elements.
