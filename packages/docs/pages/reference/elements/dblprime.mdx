# `<dblprime>`

The double prime character, used in math to indicate a second derivative. Can also be used to indicate seconds (of an angle) or inches.
Similar to a double right quote. The unicode character is &#x2033;, which is entered in HTML as `&#x2033;`.
Always used with the self-closing syntax `<dblprime/>`.
